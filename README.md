# CampusConnect
## Project Pitch
CampusConnect is a web based application that lets students track the campus shuttle. This website will allow students to see shuttle location, estimated arrival times, and route info so that they can avoid unnecessary waiting. 
## Project description 
CampusConnect will address the following problems: students will not have to wait in this cold weather in front of NAC/ 145th street subway/125th subway with unreliable or unclear shuttle schedules. This project will give them the opportunity to view live shuttle data and route details. 
## Primary User 
- students
- staff

## Features 
- Realtime tracking of busses
- interactive map showing route
- estimated arrival time
- delay notification

## Benefits 
- It will reduce waiting time
- Helps student plan their schedules
- Makes life easier for commuter student like me
